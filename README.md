JavaScript Mini-Projects Collection
Welcome to the JavaScript Mini-Projects repository! This is a curated collection of interactive, responsive, and functional front-end web applications built using HTML5, CSS3, JavaScript (DOM Manipulation), and Tailwind CSS.

These projects cover a wide range of use cases—from utility tools like a password generator to classic games and API-integrated weather applications.

🚀 Projects Included
The repository contains the following independent project folders:

1. 🔍 Dev-Detective
Description: A dynamic profile finder/search application that fetches and displays developer data (such as GitHub statistics or custom profiles).

Features: Clean UI, FontAwesome icon integration, dynamic text rendering, and responsive grid layouts.

Tech Stack: HTML5, CSS3, JavaScript (DOM API).

2. 🔢 Counter Application
Description: A classic interactive counter tool.

Features: Increment, decrement, and reset states with responsive UI feedback.

Tech Stack: HTML5, CSS3, JavaScript.

3. 🔐 Password Generator
Description: A security utility tool that generates strong, customizable passwords.

Features: Configurable length, inclusion of uppercase/lowercase letters, numbers, and special symbols, along with a "Copy to Clipboard" utility.

Tech Stack: HTML5, CSS3, JavaScript.

4. ❌ Tic-Tac-Toe Game
Description: A lightweight, interactive 2-player game built completely for the browser.

Features: Real-time win/tie detection, reset game functionality, and modern styling.

Tech Stack: HTML5, CSS3, JavaScript.

5. 🌤️ Weather App
Description: A real-time weather application displaying live atmospheric data.

Features: Dynamic UI cards, weather icon loading, and responsive layouts.

Tech Stack: HTML5, CSS3, JavaScript.

🛠️ Tech Stack & Concepts Covered
Structure & Semantics: HTML5

Styling & Themes: CSS3 (Custom properties, Flexbox, Grid layouts, and icon toolkits like FontAwesome)

Logic & Behavior: Vanilla JavaScript

Dynamic DOM Manipulation & Event Listeners

State management (including localStorage persistent layers where applicable)

Asynchronous JavaScript / API integrations
